# Bunny GIFs Placeholder

Drop your cute bunny GIF files here. The footer expects these exact filenames:

| File           | Position in footer       |
|----------------|--------------------------|
| `bunny-1.gif`  | Top-left                 |
| `bunny-2.gif`  | Top-right                |
| `bunny-3.gif`  | Bottom-left              |
| `bunny-4.gif`  | Bottom-right             |
| `bunny-5.gif`  | Middle-left              |
| `bunny-6.gif`  | Middle-right             |

## Where to find free bunny GIFs:
- https://giphy.com — search "cute bunny"
- https://tenor.com — search "bunny kawaii"
- https://ezgif.com — to resize/crop GIFs

## Until you add the files:
A translucent SVG fallback bunny is shown in each position automatically.
No code changes needed after dropping the files in this folder.
